# Muhammad Abdoel — Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.

**Live site:** [surtec.github.io/portfolio](https://surtec.github.io/portfolio)

## Pages

- **Home** — intro and hero section
- **About** — bio, education, experience, and skills
- **Work** — project showcase
- **Contact** — contact form

## Stack

- Static HTML/CSS/JS (no framework)
- Node.js + Express backend for the contact form
- MySQL for form submissions
- Deployed via GitHub Pages (Actions workflow)

## Run locally

1. Clone the repo
2. Open `index.html` in a browser for the frontend
3. For the contact form backend: install dependencies and start the server

```bash
npm install
npm start
```
